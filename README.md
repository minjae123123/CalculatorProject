# JAVA 계산기

프로젝트 소개
양의 정수 2개를 입력 받아 자바 콘솔에 계산을 하는 계산기 프로젝트

진행 목표
- 사칙연산 기능을 구현
- enum을 활용한 리팩토링을 진행한다.
- 예외 처리를 한다.


파일 구조


<img width="257" height="140" alt="image" src="https://github.com/user-attachments/assets/4506a94d-57e9-475f-a9b4-c30181134437" />

- App : 프로그램 실행과 사용자 입력 흐름 담당
- ArithmeticCalculator : 계산 로직 담당
- OperatorType : 연산자별 기능을 enum으로 관리

구현 내용
- 사칙연산 기능
- 연산 결과 조회 기능
- enum을 활용한 리팩토링(람다, 제네릭, 컬렉션은 아직 미흡하여 진행 x)
- 예외 처리
  
실행 환경
- Java 17
- IntelliJ IDEA
